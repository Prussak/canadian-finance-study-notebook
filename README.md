# Canadian Finance Study Notebook

## Overview
This repository contains a thematic study notebook created using **NotebookLM**. The project gathers 3 to 5 open-source text and PDF materials on introductory finance topics, specifically focused on the Canadian financial market. 

The goal of this notebook is to provide a structured **mini-study guide** that includes:

- Defined study objectives
- Strategic questions
- Variations of prompts tested with AI
- Summarized responses with references
- Glossary of key financial concepts
- Reusable prompts for future study sessions

## Purpose
The main purpose of this project is to demonstrate **active learning with AI**, combining critical thinking, source curation, and knowledge organization. By exploring finance through NotebookLM, users can:

- Quickly understand fundamental concepts
- Review and revise efficiently using AI-generated summaries
- Experiment with prompts to guide learning

## Structure
The notebook is organized as follows:

1. **Sources** – 3-5 selected text or PDF resources on introductory finance  
2. **Study Objectives** – clear learning goals for each topic  
3. **Strategic Questions** – questions designed to deepen understanding  
4. **Prompt Variations** – tested prompts and AI responses  
5. **Summaries & Glossary** – structured notes and key terms  
6. **Reusable Prompts** – prompts you can use for future review
